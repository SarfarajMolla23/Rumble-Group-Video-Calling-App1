# Group Video Calling Web Application


Technologies used:

   1. HTML
   2. Javascript 
   3. CSS (for styling)
   4. AGORA Sdk (for server)


How to run:
   
   1. Clone/Download this repository to your local system.
   2. Right click on `index.html` file and click `Open with live server`.
